
  # Asset Tracking Platform Wireframes

  This is a code bundle for Asset Tracking Platform Wireframes. The original project is available at https://www.figma.com/design/AkWh2we7SnvC88e5OY3Cxx/Asset-Tracking-Platform-Wireframes.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Deploying to Vercel

  1. Push your code to GitHub (if you haven’t already).
  2. Go to [vercel.com](https://vercel.com) and sign in with GitHub.
  3. Click **Add New…** → **Project** and import your **AssetTracker** repository.
  4. Leave the defaults (Vercel will detect Vite; build command `npm run build`, output `dist`).
  5. Click **Deploy**. Your app will be live at `https://your-project.vercel.app`.
  6. Optional: in Project **Settings** → **Domains** you can add a custom domain.
